# 📚 Book Catalog App

Приложение для поиска книг через Open Library API с отображением обложек.

## ✨ Функциональность

- 🔍 Поиск книг по ключевым словам
- 📖 Отображение названия, автора и года издания
- 🖼️ Загрузка обложек книг по ISBN
- ⚡ Простой и интуитивный интерфейс

## 🛠️ Используемые технологии

- **Язык:** C# (.NET Framework)
- **UI:** Windows Forms
- **HTTP-клиент:** HttpClient
- **Работа с JSON:** Newtonsoft.Json
- **API:** Open Library API (https://openlibrary.org/developers/api)
